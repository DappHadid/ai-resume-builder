This repository contains **AI Resume Builder**, a Next.js application that
helps users generate and format professional resumes using AI-powered
features.

## Project Overview

AI Resume Builder is a web application built with [Next.js](https://nextjs.org)
and TypeScript. It provides a simple interface for users to input their
experience, skills and preferences, then uses an AI model to craft
a polished resume. The app takes advantage of Next.js’s performance, routing,
and font optimisation (using `next/font` with the Geist family).

Key features include:

- Dynamic resume generation using AI prompts.
- Editor for customizing content before export.
- Export options (PDF, text).
- Responsive design and quick development workflow.
